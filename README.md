online-shopping-analysis
Online Shopping Behavior Analysis using Python (EDA, Correlation &amp; Chi-square Test

This project analyzes online shopping behavior using Python.
It includes:
- Data cleaning and preprocessing
- Correlation analysis
- Chi-square test
- Data visualization using graphs

 Features
- Customer Rating vs Purchase Frequency correlation
- Gender vs Product Category analysis
- Heatmaps, scatter plots, boxplots, violin plots

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

 How to Run
1. Install libraries:
   pip install pandas numpy matplotlib seaborn scipy

2. Run the script:
   python shopping_analysis.py

Dataset
Use any CSV file with columns like:
- Gender
- Product_Category
